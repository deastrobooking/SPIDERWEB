BasicSynth VST is a professional-grade synthesizer plugin with advanced sound synthesis capabilities, offering a wide range of creative sound design possibilities.

Supported Platforms:
  • Windows (VST2, 64-bit)
  • macOS (VST2, Universal Binary)
  • Linux (VST2, 64-bit)

Installation Instructions

Windows:
  1. Extract the downloaded ZIP file
  2. Copy the .dll file from the VST2/Windows folder to your VST plugins folder
  3. Typical locations: C:\Program Files\VSTPlugins or your DAW's custom VST folder
  4. Restart your DAW and scan for new plugins
macOS:
  1. Extract the downloaded ZIP file
  2. Copy the .vst bundle from the VST2/macOS folder to:
  3. System-wide: /Library/Audio/Plug-Ins/VST/
  4. User-only: ~/Library/Audio/Plug-Ins/VST/
  5. Restart your DAW and scan for new plugins
Linux:
  1. Extract the downloaded archive
  2. Copy the .so file from the VST2/Linux folder to your VST plugins folder
  3. Typical locations: /usr/lib/vst or ~/.vst
  4. Restart your DAW and scan for new plugins

Using the Standalone Tester

The package includes a standalone application that allows you to test the synthesizer without a DAW:

./test_synth play --oscillator wavetable --note 60 --duration 3.0

For complete documentation on using the tester, refer to the TESTER_GUIDE.md file included in the download package.
